# UIデザインガイドライン (docs/ui_design_guidelines.md)

## 項目

| 項目 | 詳細 | 記述の目的 (LLMが取る幅を狭め品質向上) |
|------|------|----------------------------------|
| **全体テーマ** | 長時間の学習に適したモダンでクリーンなデザイン。AIが多用しがちな紫や白のグラデーション、多すぎる絵文字は禁止する。 | 「よくある安全な実装」からの脱却を促す。 |
| **タイポグラフィ** | デフォルトのフォント（InterやRobotoなど）を使用せず、特定の学習用フォント（例：Noto Sans JP）を指定する。 | フォントの指定により、LLMがデザイン全体を統一的に構築するよう促す。 |
| **色構成** | 主要なアクション要素（ボタン、ハイライト）には、集中力を高める青または緑の系統を使用し、背景はダークモードを標準とする。 | 配色を具体的に指定することで、LLMが取るデザインの幅を狭める。 |
| **インタラクション** | 領域間（デッキ/動画/チャット）には、滑らかでレスポンシブなドラッグ&ドロップまたはリサイズ操作を実装する。 | 抽象的ではなく、具体的なインタラクションの仕様を記述する。 |

## 詳細

### 全体テーマ
- **デザイン理念**: 長時間の学習に適したモダンでクリーンなデザイン
- **禁止事項**: 紫系グラデーション、過度な絵文字、派手なアニメーション
- **目的**: 学習に集中できるUIの実現

### タイポグラフィ
- **メインフォント**: `Noto Sans JP`
- **コードフォント**: `JetBrains Mono` または `Fira Code`
- **フォントサイズ**:
  - 見出し(h1): `24px` / `font-bold`
  - 見出し(h2): `20px` / `font-semibold`
  - 本文: `16px` / `font-normal`
  - 補助テキスト: `14px` / `text-gray-400`

### 色構成（ダークモード基準）

```css
/* 背景色 */
--bg-primary: #0f172a;      /* スレートグレー（メイン背景） */
--bg-secondary: #1e293b;    /* カード・パネル背景 */
--bg-tertiary: #334155;     /* ホバー状態 */

/* テキスト色 */
--text-primary: #f1f5f9;    /* メインテキスト */
--text-secondary: #94a3b8;  /* 補助テキスト */
--text-muted: #64748b;      /* 非活性テキスト */

/* アクセントカラー */
--accent-primary: #3b82f6;  /* 青（メインアクション） */
--accent-hover: #2563eb;    /* 青（ホバー） */
--accent-secondary: #10b981; /* 緑（成功・確認） */
--accent-warning: #f59e0b;  /* オレンジ（警告） */
--accent-error: #ef4444;    /* 赤（エラー） */

/* ボーダー */
--border-default: #334155;
--border-focus: #3b82f6;
```

### Tailwind CSSクラス対応

| 用途 | Tailwindクラス |
|------|---------------|
| メイン背景 | `bg-slate-900` |
| カード背景 | `bg-slate-800` |
| メインテキスト | `text-slate-100` |
| 補助テキスト | `text-slate-400` |
| プライマリボタン | `bg-blue-500 hover:bg-blue-600` |
| 成功表示 | `text-emerald-500` |
| エラー表示 | `text-red-500` |

### レイアウト構成

```
+------------------+------------------------+------------------+
|                  |                        |                  |
|   デッキ          |      動画プレーヤー      |    チャット       |
|   (目次リスト)    |                        |                  |
|                  |                        |                  |
|   幅: 280px      |      幅: flex-1        |    幅: 320px     |
|   min: 200px     |      min: 400px        |    min: 280px    |
|   max: 400px     |                        |    max: 480px    |
|                  |                        |                  |
+------------------+------------------------+------------------+
```

### インタラクション
- **リサイズ**: `react-resizable-panels` または `allotment` ライブラリを使用
- **ドラッグハンドル**: 幅 `4px`、ホバー時に `bg-blue-500` でハイライト
- **トランジション**: `transition-all duration-200 ease-in-out`
- **スクロール**: カスタムスクロールバー（細め、`bg-slate-600`）
