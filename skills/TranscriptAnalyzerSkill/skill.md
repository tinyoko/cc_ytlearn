# TranscriptAnalyzerSkill

YouTube動画のトランスクリプトを分析し、構造化された目次（チャプター）と要約を生成するスキル。

## 入力

- `transcript`: 動画のトランスクリプト（タイムスタンプ付きセグメントの配列）
  - 各セグメント: `{ text: string, start: number, duration: number }`
- `videoTitle`: 動画のタイトル
- `videoDuration`: 動画の総再生時間（秒）

## 出力

### チャプター構造

```json
{
  "chapters": [
    {
      "title": "チャプター名",
      "startTime": 0,
      "summary": "このチャプターの要約（1-2文）"
    }
  ],
  "overallSummary": "動画全体の要約（3-5文）"
}
```

## 分析ルール

### 1. チャプター分割の基準

- **トピックの変化**: 話題が明確に変わる箇所で分割
- **最小長さ**: 各チャプターは最低30秒以上
- **最大長さ**: 各チャプターは最長10分以下（長い場合は細分化）
- **チャプター数**: 動画の長さに応じて3〜15個程度

### 2. チャプタータイトルの命名規則

- **簡潔性**: 15文字以内を目標
- **具体性**: そのセクションの内容を明確に表現
- **動詞形式**: 「〜について」「〜の説明」など
- **日本語優先**: 動画が日本語の場合は日本語で

### 3. 要約生成の基準

- **チャプター要約**: 各チャプターの主要ポイントを1-2文で
- **全体要約**: 動画の目的、主要トピック、結論を含む

## システムプロンプト

あなたはYouTube動画のトランスクリプトを分析する専門家です。
与えられたトランスクリプトを読み、以下のタスクを実行してください：

1. 動画の内容を理解し、トピックの変化点を特定する
2. 各セクションに適切なチャプタータイトルを付ける
3. 各チャプターと動画全体の要約を作成する

重要な注意点：
- タイムスタンプは秒単位で指定してください
- チャプターの開始時間は、そのトピックが始まる正確な位置を示す必要があります
- トランスクリプトにタイムスタンプ情報がある場合は、それを活用してください
- 技術的な用語は可能な限り平易な言葉で言い換えてください

## レスポンス形式

必ず以下のJSON形式で応答してください。他のテキストは含めないでください。

```json
{
  "chapters": [
    {
      "title": "string",
      "startTime": number,
      "summary": "string"
    }
  ],
  "overallSummary": "string"
}
```
